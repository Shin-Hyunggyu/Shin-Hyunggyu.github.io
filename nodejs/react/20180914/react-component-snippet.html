2. React Component Snippet

‘rc’ 를 입력하여 Component 골격을 생성한다.

"react component set": {
  "prefix": "rc",
  "body": [
    "import React, { Component } from 'react';",
    "import PropTypes from 'prop-types';",
    
    "\nconst propTypes = {};",
    
    "\nconst defaultProps = {};",
    
    "\nclass $1 extends Component {",
      "\tconstructor(props) {",
        "\t\tsuper(props);",
      "\t}",
      "\n\trender() {",
        "\t\treturn (",
          "\t\t\t<div>$1</div>",
        "\t\t);",
      "\t}",
    "}",

    "\n$1.propTypes = propTypes;",
    "$1.defaultProps = defaultProps;",

    "\nexport default $1;"
  ]
}
